How to Fix HP Printer Offline

Simple Troubleshooting Guide

Why Does HP Printer Show Offline?

An HP printer may appear offline because of network problems, cable issues, printer settings or communication errors between the printer and computer.

Step 1 - Check Power

Make sure the printer is turned on and showing no error lights.

Step 2 - Check Connection

If using Wi-Fi, ensure the printer is connected to the same network as your computer.

Step 3 - Restart Devices

Restart your printer, router and computer.

Step 4 - Set Printer as Default

Open Windows printer settings and set the HP printer as the default printer.
